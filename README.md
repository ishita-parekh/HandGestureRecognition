# HandGestureRecognition and controlling a game "Dangerous Dave" using recognised hand gestures.

[1]Coding is done in python. OpenCV library is used for computer vision implementation. Visual Studio IDE is used. 

[2]Hand gestures like closed fist, open fist, all the best and ok are recognised.

[3]Alongwith hand gestures number of fingers are also counted. 

[4]Concept of convex hull and convexity defect is used for gesture recogniton.

[5]Dangerous Dave is controlled using recognised hand gestures.

[6]Keyboard keys that are used to control the game are: up, left, right, up + left and up + right.
